# Mobile QA Checklist (Milestone 4)

## Layout / navigation
- Header stays visible and does not overlap content.
- Bottom nav is reachable and does not cover buttons or inputs.
- Safe-area padding applied (top/bottom) on devices with notches.

## Menu
- Accordion opens per day without clipped dropdowns.
- Add ingredient selector opens correctly.
- Quantity input and remove button are reachable on small screens.
- Empty meals show correctly.

## Shopping
- Cards layout readable; action buttons reachable.
- Warehouse input and "to purchase" values visible without horizontal scroll.
- Off-menu modal opens, fields are full width, and actions accessible.

## Ingredients
- Cards wrap correctly on narrow screens.
- Edit/Delete actions are reachable without overlap.
- Filters and add dialog are usable on mobile.

## Collaboration
- Invite form usable; buttons not clipped.
- Incoming/outgoing lists readable; action buttons reachable.

## Smoke
- Login, navigate to all tabs, perform one action per page without layout break.
